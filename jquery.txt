

一、插件编写
1、

jQuery.fn.myPlugin = function() {
  //在这里做你真正插件要做的事情
};
2、
更好的做法是采用下面的写法，这样就能在函数体内自由使用美元符号（$）。
代码的最前面有一个分号，这是为了防止多个脚本文件合并时，其他脚本的结尾语句没有添加分号，造成运行时错误。
;(function ($){
  $.fn.myPlugin = function (){
    //DO
  };
})(jQuery);

3、有时，还可以把顶层对象（window）作为参数输入，这样可以加快代码的执行速度和执行更有效的最小化操作。

;(function ($, window) {
  $.fn.myPlugin = function() {
    // Do 
  };
}(jQuery, window));
